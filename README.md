# FekraCode Platform

A full web platform showcasing software services, AI solutions, and project demos.  
Built to serve as a dynamic portfolio and client-facing system.

##  Features
- Dynamic project management
- Admin dashboard
- AI service integration
- Modern UI/UX

##  Tech Stack
- Laravel
- React
- REST APIs
